## Biography

Bjarne Stroustrup created C++ at Bell Labs in 1979. He continues to work on C++ evolution and serves on the ISO C++ committee.

## Daily Work

- Participating in C++ standardization
- Teaching at Columbia University
- Writing about C++ and programming
- Advocating for safer C++

## Challenges

> "C++ is a victim of its own success. Backwards compatibility with decades of code limits what we can change."

## Impact

C++ powers operating systems, game engines, databases, and performance-critical software worldwide. Modern standards continue to evolve the language.
