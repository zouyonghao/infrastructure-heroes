## Biography

Brendan Eich created JavaScript in 10 days in 1995 at Netscape. He later co-founded Mozilla and created the Brave browser.

## Daily Work

- Leading Brave Software as CEO
- Advocating for web privacy
- Speaking about JavaScript and the web
- Working on BAT cryptocurrency

## Challenges

> "JavaScript was created in 10 days, and we've been cleaning up the language for 25 years. But it's evolved beautifully."

## Impact

JavaScript is the most widely used programming language, running in every web browser and increasingly on servers and devices.
