## Biography

Tim Berners-Lee invented the World Wide Web in 1989 at CERN. He created HTTP, HTML, and the first web browser and server.

## Daily Work

- Leading the Web Foundation
- Advocating for web standards
- Working on Solid project for data sovereignty
- Advising on web governance

## Challenges

> "The web as I envisioned it, we have not seen it yet. The future is still so much bigger than the past."

## Impact

Tim created the foundation of the modern internet, enabling global communication and commerce that transforms every aspect of life.
