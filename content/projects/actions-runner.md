### Overview

GitHub Actions Runner is the application that runs jobs from a GitHub Actions workflow on a self-hosted machine.

### Importance

- Powers GitHub Actions self-hosted runners
- Foundation for custom CI/CD
- Critical for enterprise workflows
- Enables air-gapped CI

### Key Features

- Self-hosted execution
- Cross-platform support
- Runner groups
- Auto-scaling support

### Sustainability

GitHub Actions Runner is maintained by GitHub (Microsoft) with strong corporate backing.
