### Overview

Argo CD is a declarative, GitOps continuous delivery tool for Kubernetes that follows the GitOps pattern.

### Importance

- Leading GitOps tool for Kubernetes
- Foundation for declarative deployments
- Critical for cloud-native CD
- CNCF graduated project

### Key Features

- Git-based deployments
- Application health monitoring
- Multi-cluster support
- SSO integration

### Sustainability

Argo CD is a CNCF graduated project with strong community and corporate support from Akuity, Intuit, and others.
