### Overview

Bun is a fast all-in-one JavaScript runtime and toolkit, including a bundler, test runner, and Node.js-compatible package manager.

### Importance

- Fastest JavaScript runtime
- Growing Node.js alternative
- Foundation for fast tooling
- Pushing ecosystem forward

### Key Features

- Speed-focused design
- Node.js compatibility
- Built-in bundler
- Native TypeScript support

### Sustainability

Bun is backed by Oven (VC-funded) with commercial support planned. Development is active but dependent on funding.
