### Overview

Caddy is an open-source web server with automatic HTTPS. It is designed to be easy to use and secure by default.

### Importance

- Pioneered automatic HTTPS
- Simplifies web server deployment
- Foundation for secure-by-default hosting
- Growing adoption

### Key Features

- Automatic HTTPS via Let's Encrypt
- HTTP/3 support
- Simple configuration (Caddyfile)
- Extensible plugin system

### Sustainability

Caddy is developed by a small team with sponsorship funding. The project operates with limited resources despite its importance.
