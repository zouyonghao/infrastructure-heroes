### Overview

containerd is an industry-standard container runtime that manages the complete container lifecycle of its host system, from image transfer and storage to container execution and supervision.

### Importance

- Default runtime for Docker and Kubernetes
- CNCF graduated project
- Foundation for cloud container infrastructure
- Used by major cloud providers

### Key Features

- OCI-compliant container execution
- Image management
- Network and storage interfaces
- Snapshot support

### Sustainability

containerd is a CNCF graduated project with strong corporate support. It has an active community and solid funding through the foundation.
