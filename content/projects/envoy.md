### Overview

Envoy is an open-source edge and service proxy designed for cloud-native applications. It was originally built at Lyft.

### Importance

- Foundation of service mesh architectures
- Powers Istio and other meshes
- Critical for microservices communication
- Used by major cloud providers

### Key Features

- L7 proxy with extensibility
- gRPC and HTTP/2 support
- Observability features
- Dynamic configuration

### Sustainability

Envoy is a CNCF graduated project with strong community and corporate backing from Lyft, Google, and others.
