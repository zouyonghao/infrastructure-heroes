### Overview

InfluxDB is an open-source time series database designed to handle high write and query loads for time-stamped data.

### Importance

- Leading time series database
- Powers monitoring and IoT
- Foundation for metrics storage
- Critical for observability

### Key Features

- Time series optimized
- Flux query language
- High write throughput
- Downsampling and retention

### Sustainability

InfluxDB is backed by InfluxData with strong commercial support. The open-source edition remains actively developed.
