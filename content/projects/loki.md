### Overview

Grafana Loki is a horizontally-scalable, highly-available, multi-tenant log aggregation system inspired by Prometheus.

### Importance

- Cost-effective log aggregation
- Integrates with Grafana
- Foundation for cloud-native logging
- Growing adoption

### Key Features

- Label-based indexing
- LogQL query language
- Prometheus-like approach
- Multi-tenancy

### Sustainability

Loki is backed by Grafana Labs with strong commercial support and active development.
