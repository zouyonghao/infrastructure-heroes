### Overview

Neo4j is a graph database management system described as an ACID-compliant transactional database with native graph storage and processing.

### Importance

- Leading graph database
- Powers knowledge graphs
- Foundation for connected data
- Critical for relationship-heavy data

### Key Features

- Native graph storage
- Cypher query language
- ACID compliance
- Clustering support

### Sustainability

Neo4j is backed by Neo4j Inc. with strong commercial support. The community edition remains open source.
