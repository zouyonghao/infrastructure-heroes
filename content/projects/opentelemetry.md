### Overview

OpenTelemetry is a collection of APIs, SDKs, and tools to instrument, generate, collect, and export telemetry data for analysis.

### Importance

- Unified observability standard
- Powers modern observability stacks
- Foundation for vendor-neutral telemetry
- CNCF incubating project

### Key Features

- Traces, metrics, logs collection
- Vendor-neutral APIs
- Auto-instrumentation
- OTLP protocol

### Sustainability

OpenTelemetry is a CNCF project with strong corporate backing from major observability vendors and cloud providers.
