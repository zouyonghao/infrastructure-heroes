### Overview

Prometheus is an open-source systems monitoring and alerting toolkit originally built at SoundCloud. It is now a CNCF graduated project.

### Importance

- De facto standard for cloud-native monitoring
- Foundation of observability stacks
- Powers monitoring for Kubernetes ecosystems
- Used by organizations worldwide

### Key Features

- Multi-dimensional data model
- Flexible query language (PromQL)
- Pull-based metrics collection
- Built-in alerting

### Sustainability

Prometheus is a CNCF graduated project with strong community and corporate support. It has a healthy contributor ecosystem and solid governance.
