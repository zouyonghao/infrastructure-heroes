### Overview

Rook is an open-source cloud-native storage orchestrator for Kubernetes, providing the platform for running storage systems.

### Importance

- CNCF graduated project
- Powers Kubernetes storage
- Foundation for cloud-native storage
- Critical for stateful workloads

### Key Features

- Ceph integration
- Kubernetes native
- Self-managing
- Multi-storage backend support

### Sustainability

Rook is a CNCF graduated project with strong community support and corporate backing.
