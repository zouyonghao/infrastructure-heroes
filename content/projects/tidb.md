### Overview

TiDB is an open-source, cloud-native, distributed SQL database designed to support hybrid transactional and analytical processing (HTAP) workloads.

### Importance

- MySQL compatible distributed database
- Powers real-time analytics
- Foundation for cloud-native HTAP
- CNCF sandbox project

### Key Features

- MySQL compatibility
- Horizontal scaling
- HTAP capabilities
- Cloud-native architecture

### Sustainability

TiDB is backed by PingCAP with strong commercial support. The project has an active community.
