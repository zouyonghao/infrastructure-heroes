### Overview

TimescaleDB is an open-source time-series database optimized for fast ingest and complex queries, built as a PostgreSQL extension.

### Importance

- PostgreSQL compatible time series
- Powers IoT and monitoring
- Foundation for hybrid workloads
- Critical for time-stamped data

### Key Features

- Full PostgreSQL compatibility
- Automatic partitioning
- Continuous aggregates
- Compression

### Sustainability

TimescaleDB is backed by Timescale Inc. with strong commercial support.
